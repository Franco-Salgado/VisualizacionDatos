El dataset fue obtenido desde Kaggle a través del siguiente link como un archivo csv:

https://www.kaggle.com/datasets/bhanupratapbiswas/olympic-data

El dataset posee los datos de los participantes a lo largo de los Juegos Olímpicos desde 1896 hasta 2016.
Los datos que posee de cada participante son: ID, Nombre, Sexo, Edad, Altura, Peso, Equipo, NOC Region, Edición (Año - Invierno/Verano) y Año.

Sirve para el gráfico porque posee el año, el Equipo como nombre de país e ID como identificador único de cada atleta, por lo que asegura que no se repita cuando un atleta participa en más de una categoría.
